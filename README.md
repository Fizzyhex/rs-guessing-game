## Build Instructions

1. Clone this repository (e.g with git):

```sh
git clone "https://github.com/Fizzyhex/rs-guessing-game.git" 
```

2. Use `cargo` to build the project:

```sh
cd rs-guessing-game
cargo build --release
```

3. Run the executable!

```sh
./target/release/guessing_game.exe
```
